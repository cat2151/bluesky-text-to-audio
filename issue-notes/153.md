# issue mix playについて、「chord YM2151」や「Surge XT Chord」や「chord tone.js」などの表記を可能にする #153
[issues #153](https://github.com/cat2151/bluesky-text-to-audio/issues/153)

- chordが指定されていれば、chord表記から、chord2mmlライブラリを利用してMMLを生成、という処理をはさむ、ということ
- chord2mmlライブラリの生成したMMLが必ずしも YM2151 / Surge XT / Tone.js の表記にマッチしていないことがある。
  - このため、このときは新たにtextareaを追加表示して、chordプリプロセス後のmix MMLを表示、とすること。
    - そしてそこは1秒のデバウンスでuserが適切なmix MMLに書き換えてplay、を可能とすること。
