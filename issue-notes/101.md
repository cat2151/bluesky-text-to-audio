# issue 検証用に、Tone.jsとYM2151とずんだもんを1つのwavデータにmixして演奏する機能を試す #101
[issues #101](https://github.com/cat2151/bluesky-text-to-audio/issues/101)

# format
```
VOICEVOX ずんだもんなのだ;
YM2151 rc;
Tone.js rrg
```

# 実現方法
- このアプリだけのMML方言を追加で用意する
  - セミコロンでtrack分割
  - trackごとに、
    - track先頭に VOICEVOX / YM2151 / Tone.js というキーワードがあるかチェック
    - あればtrack識別子を、VOICEVOX / YM2151 / TONE_JS にセットし、キーワードを削除
    - ない場合は、先頭行のデフォルトはTone.jsで、2行目からは前の行を引き継ぐ
  - ※備忘、text再度の連結は、複雑なので、この段階では実施を禁止。検証してからの検討とする（連結時に同じtrack識別子のものだけ連結？あえて分割したいときは？独立していればYM2151の8ch制限がなくなる等のメリットがある）
- Tone.jsはオフラインレンダリングを実装することで、wavデータを取得可能にする。このモードのときはオフラインレンダリングに切り替えるということ
- それぞれをできれば別スレッドで並列にオフラインレンダリングさせる
- 取得したwavデータをmixingする
- それをTone.jsで演奏する
- 複雑なので検証用に、各処理フェーズにおいてconsole.logする
