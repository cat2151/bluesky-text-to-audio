# issue menuのhistory機能に「お気に入り」を追加する #127
[issues #127](https://github.com/cat2151/bluesky-text-to-audio/issues/127)

- historyと同じ場所に、お気に入りlistを最大20件表示する
- お気に入りからplayすると、textareaを上書きしてplayする
- お気に入りからplayしたときは、historyを更新しない
- historyに、playボタンの隣に「お気に入りに追加する」ボタンをつける
- お気に入りのlistには、「お気に入りから削除」ボタンをつける。削除するとhistoryに入る（うっかりミスをリカバーする用）
