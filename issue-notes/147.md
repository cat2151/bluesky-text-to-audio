# issue historyとお気に入りは、menuには「開く」だけにして（toggleにしない）、history欄とお気に入り欄の先頭に折りたたみtoggleボタンをつける #147
[issues #147](https://github.com/cat2151/bluesky-text-to-audio/issues/147)

- userがほしいのは、
- 最初はhistoryやお気に入りの欄がなく、シンプルな表示
- 必要に応じて、menuで、historyやお気に入りを開く。折りたたみ解除されて登場する
- 使ったあと、折りたたみたくなったら、historyとお気に入りそれぞれ個別に先頭をクリックすると、個別に折りたたみ
- 折りたたんだ状態で、折りたたみ解除したいときは、historyとお気に入りそれぞれ個別に先頭をクリックすると、折りたたみ解除
- historyと折りたたみを表示したあとは、わざわざ欄を消す用途はないので、「欄を消す操作」ナシにして、シンプル実装にする
