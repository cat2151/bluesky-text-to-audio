# issue VOICEVOXとSurge XTは、port error時に、新しく行エリアを表示し、そこでVOICEVOXダウンロードとSurge XTダウンロードの導線リンクを表示する #140
[issues #140](https://github.com/cat2151/bluesky-text-to-audio/issues/140)

- port errorとは、当該portのserverの起動チェック時に、起動していない状態だ、と判断され、エラーになった、くらいの意味である。

- userがほしいのは、鳴らすにはどんな手順をとればいいか、の導線リンクを表示、である

- VOICEVOXダウンロードの導線リンク : 
  - https://cat2151.github.io/voicevox-playground/
  - 説明：このchrome拡張は voicevox-playground と同じ技術で、ずんだもん読み上げができます
- Surge XTダウンロードの導線リンク：
  - https://github.com/cat2151/clap-mml-render-tui/blob/main/README.ja.md
  - 説明：このchrome拡張は clap-mml-render-tui のserverモードと接続して、Surge XTを演奏できます
